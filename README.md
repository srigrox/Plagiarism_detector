# Setup:
Frontend is in the `client` folder and backend is in the `server` folder.

## In /server/: 
```npm install express```

```npm install @msrvida/python-program-analysis```

```npm install cors```

## In /client/:
```npm install react```

# Running the application:
In both `client` and `server`: 
```npm run start```
